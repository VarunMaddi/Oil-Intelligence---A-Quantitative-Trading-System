# Oil-Intelligence---A-Quantitative-Trading-System
An end-to-end oil market intelligence &amp; cross-sectional trading system using AIS vessel traffic and events, EIA inventories, macro data, and OPEC events on Databricks. Developed bias-controlled LGBMRanker/XGBoost ensembles with purged walk-forward validation, achieving 1.89 Sharpe, 2.27 Fitness and 32.8% OOS return.
